    Import / Export
    Sauvegardez ou partagez vos données de voyage
    📤 Exporter ce voyage
    Télécharge un fichier .json contenant toutes les données du voyage actuellement sélectionné. Utile pour changer de navigateur ou partager un itinéraire à un ami.
    📥 Importer un voyage
    Charge un fichier .json exporté depuis cette application. Le voyage sera ajouté à votre liste sans écraser les autres.
    💾 Exporter tous les voyages
    Télécharge un fichier .json contenant l'intégralité de vos voyages. Idéal comme sauvegarde complète de l'application.
    🔗 Partager ce voyage
    Génère un lien de consultation en lecture seule. Le destinataire peut voir l'itinéraire, les vols, les hébergements et le budget, sans pouvoir modifier votre séjour. Il peut aussi choisir de l'importer dans sa propre application.
    Lien généré — copiez et partagez :
    🗑 Supprimer ce voyage
    Supprime définitivement le voyage sélectionné et toutes ses données. Cette action est irréversible — pensez à exporter d'abord si besoin.
    💡 Stockage des données
    Si vous êtes connecté, vos voyages sont synchronisés de manière sécurisée sur nos serveurs via votre compte, et accessibles depuis n'importe quel appareil.
    Si vous n'êtes pas connecté, les données restent uniquement dans le localStorage de votre navigateur — pensez à exporter régulièrement en .json pour éviter toute perte.
